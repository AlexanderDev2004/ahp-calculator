Panduan Aplikasi Sistem Pendukung Keputusan (SPK) Demice

Penggunaan Aplikasi Demice

Login dan Konfigurasi Awal

  1. Masuk ke aplikasi dengan username dan password
  2. Konfigurasikan parameter dasar sistem:
    • Kriteria keputusan
    • Bobot kriteria
    • Alternatif yang akan dievaluasi

Langkah-Langkah Penggunaan

1. Input Data
  • Masukkan alternatif-alternatif yang akan dibandingkan
  • Tentukan kriteria penilaian untuk setiap alternatif
  • Berikan nilai untuk setiap alternatif pada setiap kriteria
2. Proses Perhitungan
  • Pilih metode yang ingin digunakan (SAW, TOPSIS, WP, atau AHP)
  • Klik tombol "Proses" untuk menjalankan perhitungan
3. Analisis Hasil
  • Lihat ranking dari alternatif yang dievaluasi
  • Periksa detail perhitungan setiap metode
  • Bandingkan hasil dari berbagai metode
4. Ekspor Hasil
  • Ekspor hasil dalam format PDF atau Excel
  • Simpan data untuk penggunaan di masa depan

Rumus-Rumus Metode SPK

1. Metode Simple Additive Weighting (SAW)

rij = (xij / max xij) [untuk benefit]
rij = (min xij / xij) [untuk cost]
Vi = Σ (wj * rij)

Dimana:

  • rij = nilai rating kinerja ternormalisasi
  • xij = nilai atribut yang dimiliki setiap alternatif
  • Vi = nilai preferensi untuk setiap alternatif
  • wj = nilai bobot preferensi

2. Metode TOPSIS (Technique for Order Preference by Similarity to Ideal Solution)

rij = xij / √(Σ xij²)
yij = wj * rij
A+ = max(yij) [untuk benefit], min(yij) [untuk cost]
A- = min(yij) [untuk benefit], max(yij) [untuk cost]
Di+ = √(Σ (yij - A+)²)
Di- = √(Σ (yij - A-)²)
Vi = Di- / (Di+ + Di-)

Dimana:

  • rij = matriks keputusan ternormalisasi
  • yij = matriks keputusan ternormalisasi terbobot
  • A+ = solusi ideal positif
  • A- = solusi ideal negatif
  • Di+ = jarak alternatif ke solusi ideal positif
  • Di- = jarak alternatif ke solusi ideal negatif
  • Vi = nilai preferensi

3. Metode Weighted Product (WP)

Si = Π (xij^wj)
Vi = Si / Σ Si

Dimana:

  • Si = preferensi alternatif sebagai vektor S
  • xij = nilai kriteria
  • wj = bobot kriteria
  • Vi = preferensi relatif dari setiap alternatif

4. Metode AHP (Analytical Hierarchy Process)

aij = nilai perbandingan kriteria i dengan j
CI = (λmax - n) / (n - 1)
CR = CI / RI

Dimana:

  • aij = nilai perbandingan
  • CI = Consistency Index
  • λmax = eigenvalue maksimum
  • n = jumlah kriteria
  • CR = Consistency Ratio
  • RI = Random Index

Contoh Penggunaan

Untuk menjelaskan cara penggunaan, mari kita ambil contoh pengambilan keputusan untuk pemilihan pemasok dengan kriteria:

  1. Harga (30%)
  2. Kualitas (40%)
  3. Pengiriman (20%)
  4. Layanan (10%)

Dengan 3 alternatif pemasok: A, B, dan C.

  1. Input nilai untuk setiap pemasok pada setiap kriteria
  2. Pilih metode SAW
  3. Sistem akan menormalisasi data dan menghitung nilai akhir
  4. Hasil akan menunjukkan pemasok mana yang terbaik berdasarkan kriteria yang telah ditentukan
